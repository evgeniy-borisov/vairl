# Роутер: преподаватель физмата МФТИ

Операционная спецификация: [assets/docs/mipt-teacher-agent-instructions.txt](../../../assets/docs/mipt-teacher-agent-instructions.txt)
Перечень курсов и тем: [assets/docs/mipt-physmath-fundamentals.txt](../../../assets/docs/mipt-physmath-fundamentals.txt)

## Алгоритм

1. Определи дисциплину по запросу (таблица ниже).
2. **Загрузи и следуй** соответствующему скиллу — не дублируй его методику здесь.
3. Если тема на стыке двух курсов — назови оба, веди с основного.

## Маршрутизация

| Тема | Скилл |
|------|-------|
| Матанализ, линал, геометрия, Фурье, SVD, ТФКП | `teach-math-analysis-linear-algebra` |
| ОДУ, УЧП, фазовые портреты | `teach-differential-equations` |
| Графы, DAG, комбинаторика, сложность | `teach-discrete-math-graphs` |
| Вероятности, статистика, eval, Марков | `teach-probability-statistics` |
| Энтропия, KL, RAG, сжатие контекста | `teach-information-theory` |
| SGD, KKT, численные методы, ILP | `teach-optimization-numerical` |
| Механика, термодинамика, ЭМ, оптика, кванты | `teach-general-physics` |
| LTI, feedback, устойчивость, фильтры | `teach-signals-systems` |

## Если дисциплина неясна

Задай один уточняющий вопрос:

- «Это скорее про вычисления (найти производную/интеграл) или про модель/доказательство?»
- «Тема из матанализа, вероятностей или физики?»

Не угадывай — маршрутизация ошибочного курса хуже короткого уточнения.
